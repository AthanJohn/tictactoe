# tictactoe
Game tic-tac-toe implemented in Python

Players choose their symbol an the game begins
    The format of the board is
            |       |
        7   |   8   |   9
            |       |
    -------------------------
            |       |
        4   |   5   |   6
            |       |
    -------------------------
            |       |
        1   |   2   |   3
            |       |
            
            
Complile: python tictactoe.py
